Sample page of Complex Images with extended Descriptions

This page has three images each of which has an off-screen extended image description. When Javascript is not present the extended descriptions should be automatically accessible to AT always (i.e. the user will have no choice but to hear the descriptions). These extended image descriptions however will be coded to allow AT to navigate and escape the description to the next item on the page. When Javascript is present the extended descriptions will be hidden to AT and a toggle button to show/hide the description will be available to AT to get more information on demand.
