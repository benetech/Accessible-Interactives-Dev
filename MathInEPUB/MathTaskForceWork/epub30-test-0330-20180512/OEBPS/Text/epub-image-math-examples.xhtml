Math in EPUB Tests

Support for JavaScript and MathML math-010

The test passes if the assistive technology reports presence of all 4 MathML expressions and provides functionality to read it.

If the test passes, please provide notes about your MathML reading experience, and how well you were able to navigate in MathML. For example, did you hear the prefix "mathml alt text” and you heard the expression “(mathml alt text): x sub 1, y sub 1” or was the math expression expressed via the readers own math rulesets such as “left parenthesis x subscript 1, y subscript 1 right parenthesis” with no prefix.”

The test fails if you read/hear the prefix "image alt fallback" prior to the explanation of the math. The test also fails if you read/hear duplicate expressions, once for MathML and other for fall back. This means that JavaScript failed to work properly.

Please make a note if you are able to see both the image of the math equation as well as the mathML rendered equation, since the mathML version should be visually hidden off-screen but accessible to AT. Also note any other visual anomolies you encounter.
